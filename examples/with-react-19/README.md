# Headless Tooltip with React 19 + Tailwind CSS

This is a starter template for using [Headless Tooltip](https://github.com/rezasohrabi/headlessui-tooltip) with Vite, React 19, and Tailwind CSS v4.

## Tech Stack

- **React 19** - Latest React version
- **Vite** - Fast build tool
- **Tailwind CSS v4** - Utility-first CSS framework
- **TypeScript** - Type safety
- **headless-tooltip** - Headless UI tooltip component

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

## Features

This example demonstrates:

- Basic tooltip usage
- Custom styled tooltips with Tailwind CSS
- Different tooltip placements (top, bottom, left, right)
- Interactive tooltips
